  • The Movie Catalogue

    React.js, HTML, CSS, JavaScript

    This is a Movie Database made with React. It lets users search for any movie and generates a short description using The Movie Database (TMDB). This app is not yet complete and still is a work in progress. The final goal is to have a login page at the start so each user can save their favourite movies in a "Favourites" section. Also, this app will allow users to see general reviews of each film and add movies to their wishlist accordingly. As soon as the project is finished, this website will go live. For the moment, the code will be found on my github.

  • Todo List

    HTML, CSS, JavaScript, Node.js, Express.js, Mongoose

    This is a To Do List app where you can add new objectives to complete and chceck them out once done. You can also create new pages to your liking while keeping the same format in order to seperate your objectives into different categories. For example, I can have a page for all my schoolwork and another page for all the gym exercises I would like to complete today. From a technical perspective, the challenge in this project was to use Node.js with the Express framwork and link it to the Mongo library Mongoose (DBMS). The code for this project can be found on my github.
